## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.

## Commands

```bash
bun install              # install dependencies
bun run dev              # start dev server at localhost:4321
bun run build            # build production site to ./dist/
bun run preview          # preview the build locally
bun run sync-api         # sync API reference from packages/app
```

## API Reference Sync

The `sync-api` script reads `packages/app/docs/api-reference.md` (auto-generated by the app's `bun run docs`) and injects method/property tables into per-class reference pages between `<!-- AUTO:START -->` / `<!-- AUTO:END -->` markers. Run it manually after regenerating the app's API docs.
