# Bunwright

Monorepo for the Bunwright browser automation library and its website.

## Packages

| Package                                  | Description                                                                                       |
| ---------------------------------------- | ------------------------------------------------------------------------------------------------- |
| [`packages/app`](./packages/app)         | The Bunwright library and CLI — a TypeScript-first browser automation DSL built on `Bun.WebView`. |
| [`packages/website`](./packages/website) | Website (placeholder).                                                                            |

## Getting Started

```bash
bun install   # install all workspace dependencies
```

See [`packages/app/README.md`](./packages/app/README.md) for library usage and API docs.
